myFlix

The objective of this server-side apllication of to provide users with information on their favorite movies. User are able to sign up and create an account, update their personal information and list of their favorite movies.

Endpoint HTTP Method Request Format Response Format Use Case
/api/movies GET None JSON Returns list on all movies.
/api/movies/[title] GET None JSON Returns all stored information on the specified movie (title as specification)
/api/genre/[genre] GET None JSON Return data about a genre (description) by name/title (e.g., “Thriller”)
/api/movies/[title]/director GET None JSON Return data about a director (bio, birth year, death year) by name
